Two-dimensional admittance filter for force tracking with infinite compliance. This function receives as input the force tracking error, (desired force - actual force) and the computed reference velocity at the previous step. As output, the function computes the reference acceleration according to the admittance equation to regulate the force, with infinite compliance, without position reference. The output is given by the equation inv(M_d) * (f_d - f - D * x_dot)
